## Contributing
Feel free to suggest any changes.
1. Fork the repository
2. Clone the fork:
```
git clone <URL_HERE>
```
3. Install dependencies using:
```
npm install
```
4. Create your changes and suggest them using a pull request (don't forget to add remote)
